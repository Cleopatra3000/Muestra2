# CHLOE

Landing page en React y Vite con portada visual, control de mayoria de edad, galeria, paginas legales y acceso directo a WhatsApp.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## Estructura

- `src/sections`: bloques principales de la landing
- `src/components`: piezas compartidas
- `src/pages`: portada y paginas legales
- `src/config`: marca y contacto
- `public`: recursos estaticos
